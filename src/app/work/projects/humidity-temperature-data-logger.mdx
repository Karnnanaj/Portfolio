## Overview

Developed a environmental data logging system that monitors humidity and temperature conditions. The device features cloud synchronization, and remote access capabilities, enabling environmental tracking to protect laboratory equipment from high humidity.

## Key Features

- **Dual Sensor Monitoring**: High-precision humidity (±3% RH) and temperature (±0.5°C) sensors
- **Cloud Synchronization**: Automatic data upload to cloud database for remote access and analysis
- **Real-Time Monitoring**: Continuous measurement with configurable sampling intervals
- **Low Power Operation**: Deepsleep implemented for long operational lifetime
- **Wireless Connectivity**: WiFi integration for seamless cloud connectivity
- **User-Friendly Interface**: Simple setup and configuration via mobile app or web dashboard



## Outcome

A fully functional environmental data logging system for laboratory. Users now have complete visibility into environmental conditions with automatic cloud backup, enabling data-driven decisions for environmental management and compliance documentation. The system has proven reliable for extended deployments in various environments.
